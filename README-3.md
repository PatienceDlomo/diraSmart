# DiraSmart — AI Workplace Productivity Assistant

> *"Dira" is Setswana for "work" — DiraSmart means working smart.*

---

## Project Overview

DiraSmart is a single integrated AI-powered dashboard that helps professionals automate everyday workplace tasks. Instead of switching between five different tools, everything lives in one platform with sidebar navigation — all powered by the Claude AI API.

Built as part of the **CAPACITI AI Skill Accelerator Programme**.

## Features

| # | Feature | Description |
|---|---------|-------------|
| 01 | **Smart Email Generator** | Generates professional emails from bullet points. Supports formal, friendly, persuasive, and apologetic tones. Adapts to audience: client, manager, or team. |
| 02 | **Meeting Notes Summarizer** | Converts long messy notes into a structured summary with decisions, action items, owners, deadlines, and open questions. |
| 03 | **AI Task Planner** | Turns a flat task list into a prioritized, time-blocked daily or weekly schedule with a time-saving tip. |
| 04 | **AI Research Assistant** | Summarizes topics or articles into plain-language insights and concrete workplace recommendations. |
| 05 | **Workplace Chatbot** | A conversational AI assistant for quick questions, drafting help, and brainstorming — with session memory. |

---

## How It Improves Productivity

- **Less task-switching** — all five tools in one dashboard instead of five separate tabs
- **Faster drafting** — emails and summaries generated in seconds, ready to edit
- **Fewer missed follow-ups** — action items and deadlines extracted automatically from meeting notes
- **Better prioritization** — task planner ranks by urgency and importance with time blocks
- **Quicker research** — get a summary plus key takeaways instantly instead of reading everything from scratch

---

## Tools Used

- **Claude API** (`claude-sonnet-4-6`) — powers all AI features via the `/v1/messages` endpoint
- **HTML, CSS, JavaScript** — single-page dashboard app, no build step required
- **Prompt Engineering** — each feature uses a dedicated system prompt scoped to its specific task

---

## Responsible AI

- ⚠️ Persistent disclaimer shown on every AI screen
- ✎ All outputs are editable — drafts to refine, never auto-sent
- 🎯 Narrowly scoped system prompts per feature to reduce unpredictable output
- 👁 Visible loading and error states — failures are never silent
